# Book Tracker

**Автор:** Рябцев Дмитрий

**Краткое описание**

Консольное приложение для учёта прочитанных книг. Позволяет добавлять, редактировать, удалять книги, отменять последнее действие (Undo), просматривать историю действий, фильтровать книги по жанру или количеству страниц. Данные хранятся в файле `books.json`. Приложение реализовано на Python с использованием шаблона MVC и принципов ООП (инкапсуляция, наследование, полиморфизм).

2. Клонируйте репозиторий или скопируйте файлы проекта.
